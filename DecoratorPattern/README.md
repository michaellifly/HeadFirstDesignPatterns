# 装饰者模式
### 定义
装饰者模式动态地将责任附加到对象上。若要扩展功能，装饰者提供了比继承更有弹性的替代方案。
### 设计原则
* 类应该对扩展开放，对修改关闭
### 使用场景
* 动态的给对象添加额外的功能
* 在不想增加很多子类的情况下扩展类
